# Sample React Js

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).


## Requirements
- [Node JS](https://nodejs.org/en/download/)
- [NPM](https://nodejs.org/en/download/)
  - `Included in the Node Package`


## How to use
- Install Packages
  - `npm install`
- Build application
  - `npm run build`
- Run application
  - `npm start`
